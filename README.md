# Simple Todo App

This is a simple, interactive to-do list application built using JavaScript and object-oriented programming (OOP) principles. Users can add tasks with a due date, mark them as completed, and delete them as needed. The project focuses on clean structure, modular code, and form validation.

## Functionality

- Add new to-do items using a popup form
- Form validation with real-time error messages
- Unique ID generation for every task
- Mark tasks as completed using checkboxes
- Delete tasks from the list
- Due dates are formatted and displayed
- Form resets only after successful submission

## Technology

- **HTML5** – Page structure
- **CSS3** – Styling and layout
- **JavaScript (ES6+)** – Core logic and OOP structure
- **Modules** – Clean separation of concerns
- **UUID** – Unique ID generation
- **Git & GitHub** – Version control and deployment
- **GitHub Pages** – Live project hosting
